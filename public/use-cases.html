Real workflows developers build with ShopGraph.

Procurement Agent

An agent receives a purchase request with a supplier URL. It extracts product data, checks per-field confidence scores, auto-fills verified fields into the purchase order, and flags uncertain fields for human review.

Code Example

// MCP tool call
{
  "tool": "enrich_product",
  "arguments": {
    "url": "https://www.grainger.com/product/DAYTON-1-2-HP-Jet-Pump-5UXK1",
    "include_score": true
  }
}
// Confidence routing logic
const result = await shopgraph.enrich(url, { include_score: true });

for (const [field, confidence] of Object.entries(result.product._shopgraph.field_confidence)) {
  if (confidence >= 0.85) {
    po.setField(field, result.product[field]);  // Auto-fill
  } else if (confidence >= 0.60) {
    po.flagForReview(field, result.product[field], confidence);  // Human review
  }
  // Below 0.60: omit from PO
}
What this demonstrates: Per-field confidence routing from the Confidence Scoring feature. The agent makes field-level trust decisions instead of accepting or rejecting the entire response.

Catalog Ingestion

A nightly pipeline processes 500 supplier URLs. It uses strict_confidence_threshold to only ingest high-confidence data. Fields below the threshold are excluded server-side. Rejected URLs are logged for manual review.

Code Example

// Bulk ingestion with server-side filtering
const urls = await loadSupplierUrls();  // 500 URLs from database

for (const url of urls) {
  const result = await fetch('https://shopgraph.dev/api/enrich', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': 'Bearer sg_live_your_key'
    },
    body: JSON.stringify({
      url,
      format: 'ucp',
      strict_confidence_threshold: 0.85
    })
  });

  const data = await result.json();

  if (data.product) {
    await catalog.upsert(data.product);  // Only high-confidence fields present
  }
}
What this demonstrates: Server-side threshold filtering via strict_confidence_threshold. Combined with UCP output format for standardized catalog ingestion across any supplier.

Price Monitoring

An agent checks competitor pricing daily. It extracts prices from product URLs, compares against internal catalog prices, and alerts when a competitor's price changes by more than 5%.

Code Example

// Daily price check
const competitors = await loadCompetitorUrls();

for (const { url, internalPrice, productId } of competitors) {
  const res = await fetch('https://shopgraph.dev/api/enrich/basic', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ url })
  });

  const data = await res.json();
  const extractedPrice = data.product?.price?.amount;
  const priceConfidence = data.product?._shopgraph?.field_confidence?.price;

  if (extractedPrice && priceConfidence >= 0.85) {
    const change = Math.abs(extractedPrice - internalPrice) / internalPrice;
    if (change > 0.05) {
      await alerts.send({
        productId,
        oldPrice: internalPrice,
        newPrice: extractedPrice,
        confidence: priceConfidence,
        change: (change * 100).toFixed(1) + '%'
      });
    }
  }
}
What this demonstrates: The free-tier /api/enrich/basic endpoint for lightweight monitoring. Per-field confidence on price ensures alerts are based on reliable data. See Confidence Scoring.
